# Asterisk Top

A fork of the Adobe [Source Code Pro](https://github.com/adobe-fonts/source-code-pro/) font with the asterisk aligned to the top instead of the middle.

## Font installation instructions

* [Mac OS X](http://support.apple.com/kb/HT2509)
* [Windows](https://www.microsoft.com/en-us/Typography/TrueTypeInstall.aspx)
* [Linux/Unix-based systems](https://github.com/adobe-fonts/source-code-pro/issues/17#issuecomment-8967116)
* Bower<br/>
	`bower install git://github.com/adobe-fonts/source-code-pro.git#release`
* npm 2.x<br/>
	`npm install --fetch-only git://github.com/adobe-fonts/source-code-pro.git#release`
* npm 3.x<br/>
	`npm install git://github.com/adobe-fonts/source-code-pro.git#release`
